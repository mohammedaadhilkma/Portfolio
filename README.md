# Mohammed Aadhil | Futuristic Portfolio 🚀

Welcome to my professional portfolio website! This is an ultra-premium, futuristic, and high-performance React application designed to showcase my journey as a Computer Science Engineering student and my expertise in Full Stack Development and Cloud Engineering.

![Portfolio Preview](https://images.unsplash.com/photo-1460925895917-afdab827c52f?auto=format&fit=crop&q=80&w=1200)

## ✨ Features

- **Modern UI/UX**: Built with a sleek dark-mode, glassmorphism-inspired design system and neon accents.
- **Dynamic Animations**: Integrated with **Framer Motion** and **Three.js** for smooth scroll reveals and 3D background effects.
- **Responsive Design**: Fully optimized for all screen sizes, from mobile devices to large desktops.
- **Project Showcase**: Detailed project cards with category filtering and interactive modals.
- **Professional Sections**: Includes About, Skills, Experience, Education, and a robust Contact section.

## 🛠️ Tech Stack

- **Frontend**: React.js, Styled Components, Framer Motion, Three.js
- **Icons**: React Icons, DevIcons
- **Animation**: React Type Animation, Canvas-based particles
- **Communication**: Integrated with Formspree for reliable, direct email messaging.

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mohammedaadhilkma/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## 📧 Contact Setup

To enable the automatic contact form:
1. Create a free account at [EmailJS](https://www.emailjs.com/).
2. Get your `Service ID`, `Template ID`, and `Public Key`.
3. Update the keys in `src/components/Contact/index.js`.
